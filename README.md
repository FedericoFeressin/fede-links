# Fede Links

Este proyecto es una landing page personal diseñada para centralizar y mostrar de forma sencilla los enlaces más importantes de Federico Feressin, como GitHub, Instagram y LinkedIn.

## Motivo del proyecto

La idea principal es ofrecer una presencia digital clara y atractiva, donde visitantes puedan encontrar rápidamente el perfil profesional y las redes sociales de una persona en un solo lugar.

Es una propuesta simple, visual y moderna, pensada para compartir en redes, presentaciones o como enlace principal de contacto.

## ¿Qué incluye?

- Diseño limpio y centrado en la identidad visual personal
- Enlaces directos a redes sociales y perfil profesional
- Estilo moderno con Tailwind CSS
- Estructura ligera y fácil de mantener

## Tecnologías utilizadas

- HTML
- Tailwind CSS
- JavaScript (sin lógica compleja, solo para soporte del proyecto)
- pnpm como gestor de paquetes

## Cómo ejecutar el proyecto

1. Instala las dependencias:
   ```bash
   pnpm install
   ```

2. Compila los estilos con Tailwind:
   ```bash
   pnpm run build:styles
   ```

3. Abre el archivo index.html en tu navegador.

## Objetivo

Servir como una tarjeta digital personal compacta, funcional y visualmente atractiva para presentar a Federico Feressin de manera profesional.
